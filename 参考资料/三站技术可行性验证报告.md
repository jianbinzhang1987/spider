# 百能云芯、艾汐芯城、京东工品汇技术可行性验证报告

验证日期：2026-06-24  
验证环境：当前本地环境（Asia/Shanghai），Playwright headless Chromium + httpx  
测试型号：`STM32F103C8T6`，辅助型号：`RC0402FR-0710KL`

## 一、结论总表

| 站点 | 纯 HTTP 可行性 | Playwright 可行性 | 推荐方案 | 复杂度 |
|---|---:|---:|---|---|
| 百能云芯 icdeal.com | 不可行 | 受 WAF 滑块阻断 | Playwright + 人工/打码处理 + 国内稳定 IP；暂不建议短期实现自动化 | 高 |
| 艾汐芯城 ic-stk.cn | 不可裸调 | 可行 | Playwright 会话内拦截/调用 `Product/AjaxSearch` JSON，再解析结构化数据 | 中 |
| 京东工品汇 vipmro.com | 可行 | 可行 | HTTP 直接调用商品搜索 JSON API，Playwright 仅作接口发现/兜底 | 中 |

## 二、百能云芯验证结果

### 2.1 HTTP 与页面入口

- 首页可访问：`https://www.icdeal.com/` 返回 200。
- 首页包含多个 API 配置，例如：
  - `https://searchapi.icdeal.com`
  - `https://saleapi.icdeal.com`
  - `https://api.icdeal.com`
- 已验证热搜接口可直接访问：
  - `https://searchapi.icdeal.com/api/v1/sku_search/get_search_keywords?day=7`
  - 返回 200 JSON，但只是热搜关键词，不是商品搜索结果。

### 2.2 搜索行为

从首页搜索 `STM32F103C8T6` 后，页面跳转到：

```text
https://waf.icdeal.com/waf/verification?source=...
```

页面正文显示：

```text
正在验证您的身份，请滑动验证码
```

说明当前实现里直接访问 `https://www.icdeal.com/s/{keyword}/` 或从首页真实搜索，都会进入 WAF 滑块验证。

### 2.3 可行性判断

百能云芯不是简单的 SSR/SPA 问题，而是搜索链路前置 WAF 验证。当前参考资料中的“Playwright + 国内 IP”方向基本正确，但需要补充一个关键点：**仅 Playwright 不足以稳定自动化，必须处理滑块验证或提供人工介入机制**。

推荐实现：

1. 短期：标记为高风险站点，默认关闭自动查询。
2. 中期：Playwright 打开搜索页，遇到 WAF 时暂停等待人工滑动，通过后复用同一个 browser context。
3. 长期：接入第三方打码/滑块服务，或寻找官方/商务数据接口。

## 三、艾汐芯城验证结果

### 3.1 Playwright 入口可用

从首页输入 `STM32F103C8T6` 后跳转：

```text
https://www.ic-stk.cn/search/STM32F103C8T6.html
```

页面渲染成功，正文包含：

```text
相关结果共 44 条
STM32F103C8T6
库存量：50645
￥5.13876
货期：1-2工作日
```

### 3.2 核心接口

浏览器内捕获到商品数据接口：

```text
POST https://www.ic-stk.cn/Product/AjaxSearch?Keyword=STM32F103C8T6&ProductType=7
POST https://www.ic-stk.cn/Product/AjaxSearch?Keyword=STM32F103C8T6&ProductType=4
POST https://www.ic-stk.cn/Product/AjaxSearch?Keyword=STM32F103C8T6&ProductType=5
```

请求体示例：

```text
Supplier=&Brand=&HasStock=&ExistModels=
Supplier=&Brand=&HasStock=&ExistModels=&stockSource=10
```

浏览器会话内接口返回：

```json
{"state":"success","message":"ok","data":{...}}
```

### 3.3 裸 HTTP 验证

使用 httpx 直接 POST 同一接口，即使先访问首页和搜索页获取 Cookie，仍返回：

```json
{"state":"error","message":"检测验证，无权限访问","data":null}
```

### 3.4 可行性判断

艾汐芯城当前可稳定由 Playwright 进入搜索页，并能在浏览器会话内拿到结构化 JSON。裸 HTTP 会触发检测验证，因此不建议做纯 HTTP adapter。

推荐实现：

1. 使用 Playwright 打开 `https://www.ic-stk.cn/search/{mpn}.html`。
2. 拦截 `Product/AjaxSearch` 响应，优先解析 JSON 中的 `ProductList` / `ChannelList`。
3. DOM 解析只作为兜底，不建议继续用正则扫整页 HTML。

## 四、京东工品汇验证结果

### 4.1 地域限制状态

当前本地环境访问京东工品汇未出现参考资料中的 HTTP 493：

- 首页 `https://www.vipmro.com/` 返回 200。
- 搜索页 `https://www.vipmro.com/ss?keyword=STM32F103C8T6` 可正常渲染。

搜索页正文包含：

```text
共 29 件相关商品
型号：STM32F103C8T6
￥7.945606 /片
￥6.619812 /片
```

### 4.2 核心接口

Playwright 捕获到核心商品搜索接口：

```text
GET https://www.vipmro.com/interface1/goods/search/mall/v2/1/20
```

关键参数：

```text
keyword=STM32F103C8T6
searchType=3
platform=2
channel=2
```

httpx 直接调用同一接口成功：

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "request": {"keyword": "STM32F103C8T6"},
    "resTotal": 29,
    "goodsList": [
      {
        "model": "STM32F103C8T6",
        "brandName": "ST",
        "id": "82500778"
      }
    ]
  }
}
```

后续库存/发货接口：

```text
POST https://www.vipmro.com/interface1/goods/stockPromiseInfoWithNum
```

页面还会请求促销、标签、组合信息等接口，但基础比价所需的型号、品牌、商品 ID、搜索总数和部分价格信息已经能从搜索接口或页面 JSON 获得。

### 4.3 可行性判断

京东工品汇应从“暂不实现/极高难度”调整为“HTTP 优先，可实现”。需要保留地域限制兜底逻辑，因为在海外节点仍可能返回 493。

推荐实现：

1. HTTP adapter 直接请求 `/interface1/goods/search/mall/v2/1/20`。
2. 解析 `data.goodsList`，按 `model` 精确匹配优先。
3. 如搜索接口缺少库存/发货字段，再批量调用 `stockPromiseInfoWithNum`。
4. 遇到 HTTP 493 或 `x-jfe-reason: deny:geo` 时返回“需国内 IP”。

## 五、对现有代码的调整建议

当前实现文件：

- `src/adapters/icdeal.py`
- `src/adapters/icstk.py`
- `src/adapters/vipmro.py`

建议调整：

1. `IcdealAdapter`：保留 Playwright，但增加 WAF 滑块检测，遇到 `waf.icdeal.com/waf/verification` 时明确返回 `FAILED: WAF滑块验证`，不要误判为 `NOT_FOUND`。
2. `IcstkAdapter`：从 HTML 正则解析改为监听 `Product/AjaxSearch` 响应，解析 JSON。
3. `VipmroAdapter`：改成 HTTP adapter；Playwright 只作为失败兜底。搜索 URL 应从 `/search?keywords=` 调整为 `/ss?keyword=` 或直接调用 `/interface1/goods/search/mall/v2/1/20`。

## 六、最终判断

- 百能云芯：技术上可人工辅助访问，但全自动采集不可稳定保证，短期不建议列为可交付站点。
- 艾汐芯城：技术可行，建议用 Playwright 会话内 API 拦截实现。
- 京东工品汇：技术可行，且当前环境可纯 HTTP 获取结构化搜索结果，应优先实现 HTTP adapter。
